Simple Blog - Blog programming example

Hi! This site purely exists as an example on how to write a super simple blog system in different programming languages. At the top, you should be able to select different programming languages and hopefully, this page and post will look exactly the same.

Requirements

I wanted to make an extremely basic example of how this blog would look in different languages, so the requirements are quite straightfoward;

  • Home page should contain a top navigation bar and it should include;
    • Title
    • Links to the different programming language variants of this blog
  • Home page should show the latest blog post, consisting of;
    • Title
    • Author
    • Unix timestamp
    • Content
  • Home page should have a section on the right of the blog post, containing;
    • A list of the previous posts
  • The content of the post should support Markdown
  • There should be a simple HTML footer with some fake copyright information

Furthermore, there should be a page where you can create a new post. It is also good to indicate that this page is implemented on each of the languages and should be the same across all of them.

Tech

All sites run on separate Azure App Services in Docker containers. See the respective repositories for examples of this and explanation of the code.

Disclaimer

Please don't use this on anything even remotely close to production. In fact, this code should not be in the vicinity, or even in earshot of any production server.
